# LLM Chatbots Repository

This repository contains a collection of chatbot projects built using Large Language Models (LLMs). Each project explores different use cases of conversational AI, ranging from document summarization and question answering to custom RAG (Retrieval-Augmented Generation) systems and domain-specific assistants.

## What's Inside

* Document & Terms/Conditions Summarizers
* RAG-based Chatbots
* Local LLM Applications
* Knowledge Base Assistants
* Experimental AI Chat Interfaces
* Prompt Engineering Examples

## Purpose

The goal of this repository is to learn, experiment with, and build practical applications using modern LLM technologies while understanding the underlying concepts behind AI-powered chat systems.

## Technologies

Projects in this repository may use:

* Python
* Ollama
* LangChain
* Streamlit
* Vector Databases
* Open-Source LLMs (Mistral, Llama, etc.)

## Getting Started

Each project contains its own setup instructions and dependencies. Navigate to the respective project folder and follow the project-specific README for installation and usage.
